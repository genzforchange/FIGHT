How It Works

Set Up

Download the Apple Shortcut here and press “Set Up Shortcut”. Carefully follow the instructions on the screen. NOTE: This only works on Apple devices.

You will be prompted to add your name to the email signature, then accept requests to receive notifications and allow the tool to send emails on your behalf. You must accept these requests to use the tool.

Create a Daily Automation

  1. Create a new automation:

    Tap the Automation icon at the bottom of your screen, then hit the to start a new one.

  2. Select “Time of Day” trigger:

    Set it to run daily, then press next at the top right corner.
    Optionally enable “Run Immediately” so you don’t have to confirm each day.

  3. Select the Shortcut

    Once you do this, it should auto-save and take you back to the Automation menu. You're good to go!
